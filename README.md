# Agent Memory Store

Persistent memory storage for AI agents with retrieval and decay features.

[![CI](https://github.com/Retsumdk/agent-memory-store/workflows/CI/badge.svg)](https://github.com/Retsumdk/agent-memory-store/actions)
[![TypeScript](https://img.shields.io/badge/TypeScript-3178C6.svg)](https://www.typescriptlang.org/)
[![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)](LICENSE)

## Features

- **Stable Memory Management** - CRUD operations for memories with metadata
- **Vector Retrieval** - Context-based search for relevant memories
- **Decay Mechanisms** - Age-based decay for least-recent memories
- **Session Persistence** - Maintain context across agent restarts

## Installation

```bash
npm install agent-memory-store
```
